Question 1: SQL Queries in Access

(A) What is the average household size for low-income households in the MAPC region?

Note that HHSIZ is an attribute in the HH table (0.5 point)

Code:

SELECT AVG(HHSIZ) AS avg_household_size
FROM HH
WHERE INCOME < 4;

Output:

avg_household_size
1.69566457501426

(B) What is the average age of women versus men in low-income households in the MAPC region where the person reported their age (exclude records when AGE=99 or 0)?

GENDER and AGE are in the PERS table (0.5 point)

Code:

SELECT
    PER.GEND,
    AVG(PER.AGE) AS average_age
FROM HH, PER
WHERE (HH.SAMPN = PER.SAMPN)
    AND (HH.MPO = "MAPC")
    AND (HH.INCOME < 4)
    AND (PER.AGE <> 99 AND PER.AGE <> 0)
GROUP BY PER.GEND;

Output:

GEND average_age
1 46.6310043668122
2 50.0327635327635

(C) What is the average trip duration for low-income women versus low-income men living in the MAPC region who reported their trip duration?

TRPDUR is in the trips table PLACE ‚Äì you can use low-income trips in MAPC that you imported for this. You must use a group by query for this and exclude trips which had a duration of 0. (1 point)

Code:

SELECT GEND AS PersonGender, AVG(TRPDUR) AS AverageTripDuration
FROM LowincomeMAPCtrips
WHERE TRPDUR > 0
GROUP BY GEND;

2 15.7362163768762

(D) What percentage of trips made by low-income households in MAPC region were by transit Bus, Subway, Walk, Bike, Auto driver and Auto passenger?

MODE is an attribute in the trips table trips table PLACE ‚Äì you should use low-income trips in MAPC that you imported for this query. You must use a group by query. You can use Excel to calculate the percentages after generating counts for each mode. (1 points)

Code:

SELECT
    IIF(MODE = 1, 'Walk',
    IIF(MODE = 2, 'Bike',
    IIF(MODE = 3, 'Auto Driver',
    IIF(MODE = 4, 'Auto Passenger',
    IIF(MODE = 5, 'Transit Bus',
    IIF(MODE = 6, 'Subway', 'Other')))))) AS TransportationMode,
    COUNT(*) AS TripCount,
    ROUND((COUNT(*) * 100.0 / (SELECT COUNT(*) FROM LowincomeMAPCtrips)), 2) AS PercentageOfTrips
FROM LowincomeMAPCtrips
GROUP BY IIF(MODE = 1, 'Walk',
    IIF(MODE = 2, 'Bike',
    IIF(MODE = 3, 'Auto Driver',
    IIF(MODE = 4, 'Auto Passenger',
    IIF(MODE = 5, 'Transit Bus',
    IIF(MODE = 6, 'Subway', 'Other'))))))
ORDER BY COUNT(*) DESC;
